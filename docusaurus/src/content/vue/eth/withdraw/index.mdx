# Configure the Coinflow Withdraw Component

Add the `CoinflowWithdraw` component into your UI and configure the props with the following information:

- `wallet` - The Ethereum Wallet
- `merchantId` - Your Merchant ID
- `env` - Use `sandbox` for testing and `prod` for production
- `blockchain` - The Blockchain you are using (in this case, `eth`)
- `onSuccess` - (optional) A callback function that is called when the withdraw is successful
- `email` - (optional) Email of the user that will be prefilled for them
